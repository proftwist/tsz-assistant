# Чат-бот для ТСЖ

## Общее описание

Это итоговая работа студента Владимира Бычко, группа ПИ-6-Р.
Проект представляет собой веб-приложение чат-бота, предназначенное для взаимодействия с жителями дома через интерфейс чата. Бот отвечает на различные вопросы, связанные с домом, коммунальными услугами, правилами проживания и другой информацией о ТСЖ (Товарищество Собственников Жилья).

## Функциональность

- Интерактивный чат-интерфейс для общения с ИИ-ассистентом
- Ответы на вопросы о доме, коммунальных услугах и правилах ТСЖ
- Современный пользовательский интерфейс с анимациями
- Адаптивный дизайн для различных устройств
- Индикатор загрузки при обработке запросов

## Архитектура приложения

### Фронтенд
- **HTML/CSS/JavaScript**: Современный интерфейс чата с анимациями
- **Шаблонизатор Jinja2**: Для рендеринга HTML-страниц с историей чата
- **Асинхронные запросы**: Используется Fetch API для взаимодействия с бэкендом

### Бэкенд
- **Flask**: Веб-фреймворк Python для обработки HTTP-запросов
- **Маршруты**:
  - `GET /` - отображение главной страницы с чатом
  - `POST /chat` - обработка сообщений пользователя и генерация ответов
- **Модели данных**: Хранение истории чата и другой информации

### Структура проекта
```
├── app.py              # Главное приложение Flask
├── models.py           # Модели данных
├── requirements.txt    # Зависимости проекта
├── prompts/            # Промпты для генерации ответов
│   └── prompt1.txt     # Файл с базой промптов
├── static/             # Статические файлы (CSS, JS)
│   ├── style.css       # Стили интерфейса
│   └── script.js       # Клиентский JavaScript
└── templates/          # HTML-шаблоны
    └── index.html      # Главная страница чата
```

## Особенности реализации

### Фронтенд
- Современный дизайн с градиентами и анимациями
- Плавные переходы и эффекты без использования фреймворков
- Адаптивная верстка для мобильных устройств
- Индикатор загрузки при обработке запросов
- Поддержка отправки сообщений по клавише Enter

### Бэкенд
- Обработка сообщений от пользователя
- Генерация ответов от имени ТСЖ
- Хранение истории чата
- REST API для взаимодействия с фронтендом
- Весь код, питон и статика, прокомментирован на русском языке

### Скрипты
- Bash-скрипт clear-history.sh для очистки бд с историей сообщений.
- Если в системе нет утилиты sqlite3, скрипт запустит brew, скачает и инсталлирует её.

## Установка и запуск

1. Установите рабочее окружение:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python app.py
```

3. Откройте в браузере http://localhost:5000

## Использование

1. Введите ваш вопрос в поле ввода внизу чата
2. Нажмите «Отправить» или клавишу Enter
3. Дождитесь ответа от ИИ-ассистента ТСЖ
4. Продолжайте диалог при необходимости

## Технологии

- **Python** - основной язык программирования
- **Flask** - веб-фреймворк
- **HTML5/CSS3/JavaScript** - фронтенд технологии
- **Jinja2** - шаблонизатор
- **Fetch API** - для асинхронных запросов
- **ShellScript** – для скриптов.
